2.Estándar W3C: Se convirtió en estándar recomendado en junio de 2012 y es fundamental para el diseño web adaptativo.
3.Soporte del Navegador: Los navegadores tienen diferentes versiones que especifican el soporte para la regla `@media`.
4.Tipos de @media CSS:
-all: Para todos los tipos de medios.
-print: Para impresoras.
-screen: Para pantallas de ordenadores, tabletas, teléfonos inteligentes, etc.
-speech: Para lectores de pantalla que leen la página en voz alta.
5.Ejemplos de @media queries:
-Computadora de escritorio: Estilo normal.
-Laptop y tablet horizontal: `@media screen and (max-width: 1024px)`
-Tablet vertical y smartphone horizontal**: `@media screen and (max-width: 720px)`
-Smartphone vertical**: `@media screen and (max-width: 480px)`


Notas


Las media queries son un modulo de CSS3 que adapta la representacion del contenido a las caracterirsticas de un dispositivo. Se conviertio en un estandar recomendado por la W3C en junio de 2012 y es un principio basico del diseño web adaptativo.
